# YouTube Automation

YouTube Automation is written in Python and Selenium

## Requirements

Look at your web brower version and download the version as
same as driver

### Linux and masOs

Prerequisites:

* [Python][Python-download]
* [Chrome][Chrome-download]
* [Firefox][Firefox-download]

1. Run:

        $ python3 main.py

### Windows

Prerequisites:

* [Python][Python-download]
* [Chrome][Chrome-download]
* [Firefox][Firefox-download]

1. Run:
    
        $ python main.py
